# RadioDumper

Radio recorder

## Installation

Add this line to your application's Gemfile:

    gem 'radio_dumper'

And then execute:

    $ bundle

Or install it yourself as:

    $ rake build
    $ rake install

## Usage(Example)

    $ radio_dumper -p radiko -t 30 -d ~/ --channel TBS

## Dependencies

* rtmpdump
* swftools
